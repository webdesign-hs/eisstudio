# Eisstudio – Die Eismanufaktur Prada

Moderne, cineastische Website für eine italienische Eismanufaktur.

## Features
- Filmstudio-Design mit Parallax-Effekten
- Scroll-Snapping zwischen Sektionen
- Responsive Design
- Smooth Animations
- Hero Push-Up Effekt

## Struktur
- `index.html` - Hauptseite
- `style.css` - Styling & Animationen
- `script.js` - Interaktionen & Parallax
- `img/` - Bilder (Hero, Akte, Eissorten)

## Sektionen
1. **Hero** - Filmplakat-Style mit Scroll-Pfeil
2. **Über uns** - 3 Akte der Firmengeschichte
3. **Sortiment** - Eissorten als Filmplakate
4. **Kontakt** - Minimalistisch mit Social Links

## Technologie
Vanilla HTML/CSS/JavaScript - keine Frameworks
